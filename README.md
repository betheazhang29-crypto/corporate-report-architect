# 企业报告构建专家

专业的企业报告生成技能，生成包含10+专业图表和深度分析文字的Word尽调报告（15-20页，1.7MB+）。

## 功能特点

- **12种专业图表**：营收趋势、利润分析、现金流、资产负债、行业对比、SWOT雷达、风险矩阵等
- **深度分析文字**：不是简单列表格，而是2-3页的深入分析
- **8个核心章节**：企业基本情况、行业分析、经营情况、财务分析、融资情况、担保情况、风险分析、授信建议
- **银行标准格式**：仿宋字体、1.15倍行距、10磅段后间距

## 使用方法

```
使用 corporate-report-architect 技能生成【某某公司】的尽调报告
```

## 输出

- `尽调报告_{company_name}_{date}.docx`：完整的Word尽调报告（15-20页，1.7MB+）

## 前提条件

需要已有 `intelligence_data.json` 和 `analysis_results.json`。

## 独立使用

本技能可以独立使用，也可以作为完整尽调流程的一部分。

## 安装

在 Manus 技能广场搜索 "corporate-report-architect" 并安装。
